API Refuge - Gestion des Chats (1.0.0)

Download OpenAPI specification:

API REST permettant de gérer les chats du refuge. Les routes POST / PUT / DELETE sont protégées par un jeton JWT.

Authentification

Générer un jeton JWT

Retourne un jeton JWT pour un chat si le login est valide. Identifiants : numeroDossier + nom.

Request Body schema: application/json
required
numeroDossier
required
number
nom
required
string

Responses

Serveur local (développement)

http://localhost:3000/api/auth/generatetoken

Request samples

  • Payload
Content type
application/json
{
  • "numeroDossier": 3392,
  • "nom": "Muscade"
}

  • 401

}

Chats

Lister tous les chats

Retourne la liste complète des chats du refuge. Possibilité d’appliquer des filtres via la query string.

query Parameters
disponible
boolean

Filtrer par disponibilité (true / false)

tauxEnergie
number [ 1 .. 5 ]

Filtrer par taux d'énergie

race
string

Filtrer par race

Responses

Serveur local (développement)

http://localhost:3000/api/chats

Response samples

  • 200
  • 500
Content type
application/json
{
  • "success": true,
  • "data": [
